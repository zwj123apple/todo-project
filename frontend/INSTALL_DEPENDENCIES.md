# 前端依赖安装指南

## 需要安装的依赖

### Day 5: React Query + 虚拟列表

```bash
npm install @tanstack/react-query
npm install @tanstack/react-virtual
```

### Day 6: 拖拽排序

```bash
npm install @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities
```

### Day 9: 图表库

```bash
npm install recharts
```

### 其他工具库

```bash
npm install clsx
npm install date-fns
```

## 一键安装所有依赖

```bash
npm install @tanstack/react-query @tanstack/react-virtual @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities recharts clsx date-fns
```

## 安装后需要做的配置

1. **React Query配置** - 创建QueryClient Provider
2. **类型定义更新** - 添加新的类型
3. **组件创建** - 任务卡片、拖拽容器等

详见各Day的开发文档。
