# Day 6: 拖拽排序功能实现指南

## 📦 安装依赖

```bash
npm install @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities
```

## 🎯 功能特性

1. **拖拽排序任务**
   - 在同一列表内拖拽
   - 跨列表拖拽（改变状态）
2. **乐观更新**
   - 拖拽立即更新UI
   - 后端失败自动回滚

3. **视觉反馈**
   - 拖拽时的透明度变化
   - 放置位置的指示器
   - 平滑动画过渡

4. ⏳ 实现拖拽排序逻辑

## 🚀 技术亮点
